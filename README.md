# Soft Graviton Production

* Numerical computations for graviton production from inflaton $n$-body decay: evaluating collision terms with multidimensional phase-space integral, and the gravitational-wave spectra beyond the soft limit.
* RAMBO Monte Carlo algorithm for the soft phase factor $\mathcal{J}^{(n)}_{\text{soft}}$ for inflaton $n$-body decay.
